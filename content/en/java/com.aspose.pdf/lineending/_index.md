Enumerates the line ending styles to be used in drawing the line.

## Fields {#fields}

| Field | Description |
| --- | --- |
| [None](#None) | No line ending. |
| [Square](#Square) | A square filled with the annotation’s interior color, if any. |
| [Circle](#Circle) | A circle filled with the annotation’s interior color, if any. |
| [Diamond](#Diamond) | A diamond shape filled with the annotation’s interior color, if any. |
| [OpenArrow](#OpenArrow) | Two short lines meeting in an acute angle to form an open arrowhead. |
| [ClosedArrow](#ClosedArrow) | Two short lines meeting in an acute angle as in the OpenArrow style and connected by a third line to form a triangular closed arrowhead filled with the annotation’s interior color, if any. |
| [Butt](#Butt) | A short line at the endpoint perpendicular to the line itself. |
| [ROpenArrow](#ROpenArrow) | Two short lines in the reverse direction from OpenArrow. |
| [RClosedArrow](#RClosedArrow) | A triangular closed arrowhead in the reverse direction from ClosedArrow. |
| [Slash](#Slash) | A short line at the endpoint approximately 30 degrees clockwise from perpendicular to the line itself. |
